<h1>Personal Portfolio Wesbite</h1>

This is a personal portfolio website created with React, Material UI and many other technologies. It includes several sections and some cool JavaScript animation. This portfolio was created to showcase personal projects.

<h2>Technologies Used</h2>

ReactJS, Context API, Material UI, Framer Motion, React Icons, React Leaflet, EmailJS, SwiperJS, React Parallax Tilt, React Lottie, React Hook Form, Yup Schema Builder etc.
